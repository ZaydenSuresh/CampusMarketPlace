# CampusMarketPlace

Campus Marketplace is a university marketplace web application developed for the Software Design Project at the University of the Witwatersrand. The platform allows students to buy and sell items within the campus community through a secure and user- friendly system.

--- 

# Features
- User authentication and account management
- Create, edit, and manage listings
- Upload listing images
- Browse and search listings
- Seller ratings and reviews
- Transaction and purchase tracking
- Seller analytics dashboard
- Admin moderation tools
- Tech Stack

--- 

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database & Storage
- Supabase

--- 

# Prerequisites

### Note: Important information on software architecture, proof of meetings and sprint documentation can be found under the "docs" folder

--- 

### Setup Requirements:
- Before running the application, ensure you have the following installed:
  - Node.js
  - npm

 

- Clone the Repository:
  - git clone https://github.com/ZaydenSuresh/CampusMarketPlace.git
  - cd CampusMarketPlace

- Install Dependencies:
  - npm install

--- 

### Running the Application:
- Start the development server:
  - npm run dev
  - If npm run dev is unavailable, use: npm start

--- 

### Accessing the Application

- Once the server is running, open:
  - http://localhost:3000
  - Environment Configuration

***The repository already includes the required environment configuration for the project.***
